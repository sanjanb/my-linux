### 📏 Step 1: Partitioning your D-Drive

You should perform this **inside Windows** before you even touch the USB drive.

**How much space?**

* **Minimum:** 100 GB (Good for a basic dev environment).
* **Recommended:** **256 GB - 500 GB**. (AI/ML models and datasets are huge. A single LLM like Llama-3 can take 5–10 GB, and you'll want space for your `.venv` environments).

**Action:**

1. Right-click the **Start Button** > **Disk Management**.
2. Right-click your **D: Drive** (or whichever drive has space) and select **Shrink Volume**.
3. Enter the amount to shrink (e.g., `256000` for ~250 GB).
4. Click **Shrink**. You will now see a black block labeled **"Unallocated Space"**. **Leave it as it is.** Do not format it.

---

### 💿 Step 2: BIOS & Booting

1. Plug in your Ventoy USB with the `cachyos-desktop-linux-260308.iso`.
2. Restart your ROG Strix and mash **F2** (or **Del**) to enter BIOS.
3. **Settings to Change:**
* **Secure Boot:** Disabled (or set to "Other OS").
* **Fast Boot:** Disabled.
* **VMD Controller:** If your laptop has this enabled, Linux might not see your SSD. You may need to disable it (caution: Windows might need a registry fix if you change this later).


4. Press **F8** for the Boot Menu and select the **UEFI: USB Drive** (Choose the **NVIDIA** driver option in the CachyOS menu).

---

### 🛠️ Step 3: CachyOS Installer Settings (Step-by-Step)

Once the Live Desktop loads, click **Launch Installer**.

1. **Welcome/Location:** Choose your Language and Timezone.
2. **Partitions (CRITICAL):**
* Select **Manual Partitioning**.
* Find the **Unallocated Space** you created in Windows.
* **Create Root Partition:** Click the unallocated space > **Create**.
* **Size:** Use all remaining space.
* **File System:** **BTRFS**.
* **Mount Point:** `/` (slash).
* **Flags:** Check `root`.


* **The EFI Partition:** CachyOS 2026 usually detects your existing Windows EFI partition. Select it (it’s usually ~100-500MB and FAT32), click **Edit**, set Mount Point to `/boot/efi`. **DO NOT FORMAT IT**, or Windows won't boot.


3. **Desktop Environment:**
* Select **Hyprland**.
* **Crucial for HyDE:** Uncheck the "CachyOS Hyprland Settings" or "Customization" boxes if they appear. You want a **Vanilla Hyprland** install so that the HyDE script can apply its own styles without conflicts.


4. **Bootloader:**
* Select **Limine** (recommended in 2026 for CachyOS) or **GRUB**. If using Limine, it handles Windows detection very well.


5. **Kernel:** Choose the default **Linux-CachyOS**.
6. **Drivers:** Ensure **NVIDIA** is selected.

---

### 🎨 Step 4: Installing HyDE Configs

After the installation completes and you reboot into the new OS:

1. Open the terminal (Kitty).
2. **Clone and Install HyDE:**
```bash
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh

```


3. Follow the prompts. Choose the **NVIDIA** options when asked.
4. Once finished, press `Super + Shift + T` to pick your first theme.

---

### 🏁 Final Dual-Boot Check

After installing, if you don't see Windows in the boot menu:

* Open terminal and run: `sudo os-prober` then `sudo grub-mkconfig -o /boot/grub/grub.cfg` (if using GRUB).
* If using Limine, it usually detects Windows automatically on the next restart.
