Creating a personal **Dotfiles** repository on GitHub is the final step in turning your setup into a portable, professional environment. It ensures that if you ever get a new laptop or need to reinstall, you can be back to your exact workflow in minutes.

We will use **GNU Stow**. It is a "symlink manager"—it makes Linux think your config files are in their normal folders (like `~/.config`), while they actually sit inside your GitHub folder.

---

### 📂 1. Structure Your Dotfiles Repository

First, create a folder to hold everything. We want a structure where each app has its own folder.

```bash
mkdir ~/dotfiles
cd ~/dotfiles
mkdir -p hypr kitty waybar nvim zsh scripts

```

### 🏗️ 2. Move Your Existing Configs

Instead of copying, we **move** the files into the `dotfiles` folder. Stow will link them back later.

```bash
# Move Hyprland and HyDE configs
mv ~/.config/hypr/* ~/dotfiles/hypr/
mv ~/.config/kitty/* ~/dotfiles/kitty/
mv ~/.config/waybar/* ~/dotfiles/waybar/

# Move your shell config
mv ~/.zshrc ~/dotfiles/zsh/.zshrc

```

### 🔗 3. Use Stow to "Link" Everything

Install Stow: `sudo pacman -S stow`

Now, you "stow" the folders. This creates a symbolic link from your `~/dotfiles` back to where the apps expect them.

```bash
cd ~/dotfiles
stow hypr
stow kitty
stow waybar
stow zsh

```

*Note: If you see an error that a file already exists, delete the empty file in `~/.config` first, then run the stow command again.*

---

### 🚀 4. Push to GitHub

1. Create a new repository on GitHub named `dotfiles`. **Keep it Private** if you have any personal API keys or names in your scripts!
2. Connect your local folder to GitHub:

```bash
git init
git add .
git commit -m "Initial commit of ROG Strix CachyOS dotfiles"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/dotfiles.git
git push -u origin main

```

---

### 🛠️ 5. The "Fresh Install" Recovery Command

Next time you install CachyOS, you don't need to manually configure anything. You just do this:

```bash
git clone https://github.com/YOUR_USERNAME/dotfiles ~/dotfiles
cd ~/dotfiles
sudo pacman -S stow
stow hypr kitty waybar zsh

```

**Boom.** Your entire desktop, keybinds, and themes are back instantly.

---

### ⚠️ Final Warning: Security

**Check your `.zshrc` or `.env` files before pushing!**
AI interns often have API keys for OpenAI, Hugging Face, or Kaggle.

* **Do not** push these to a public repo.
* Create a `.gitignore` file in your `~/dotfiles` and add any file that contains a password or token.

```bash
echo ".env" >> ~/dotfiles/.gitignore
echo "secrets.sh" >> ~/dotfiles/.gitignore

```
